# 股票战法指南

专业的股票短线与趋势交易战法详解网站，使用 VitePress 构建。

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173 查看网站

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 项目结构

```
stock/
├── docs/                  # 文档目录
│   ├── index.md          # 首页
│   ├── short-term/       # 短线战法
│   │   └── index.md
│   └── trend/            # 趋势战法
│       └── index.md
├── .vitepress/           # VitePress 配置
│   └── config.ts
├── package.json
└── README.md
```

## 内容说明

- **短线战法**：包含8种专业短线交易战法
- **趋势战法**：包含8种趋势跟踪战法

每个战法都包含概述、操作要点、案例和风险提示。
