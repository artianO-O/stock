# A股板块月度涨幅分析工具

基于 Baostock 数据，分析并可视化展示每月涨幅前3的行业板块。

## 📁 文件结构

```
analysis/
├── fetch_sector_data.py   # 数据获取脚本
├── index.html             # 可视化页面
├── requirements.txt       # Python依赖
├── data/                  # 数据目录（运行脚本后生成）
│   └── sector_analysis.json
└── README.md
```

## 🚀 使用方法

### 1. 安装依赖

```bash
cd analysis
pip install -r requirements.txt
```

### 2. 获取数据

```bash
python fetch_sector_data.py
```

⚠️ **注意**：首次运行可能需要较长时间（约30-60分钟），因为需要遍历所有行业的股票数据。

### 3. 查看可视化页面

方式一：直接用浏览器打开 `index.html`（会使用内置示例数据）

方式二：启动本地服务器（推荐，可加载真实数据）

```bash
# 使用 Python 内置服务器
python -m http.server 8080

# 然后访问 http://localhost:8080
```

## 📊 功能特性

- 🎯 **年份筛选**：支持按年份（2023/2024/2025）筛选查看
- 📈 **多维度图表**：
  - 柱状图：每月Top3板块涨幅对比
  - 饼图：板块上榜次数排行
  - 折线图：月度最强板块涨幅趋势
- 📋 **详细表格**：完整的月度排行榜数据
- 🎨 **深色主题**：专业金融风格界面

## 📌 数据说明

- **数据来源**：Baostock（证券宝）
- **行业分类**：按申万行业分类标准
- **涨幅计算**：月度涨幅 = (月收盘价 - 月开盘价) / 月开盘价 × 100%
- **复权方式**：前复权
