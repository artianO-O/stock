# 怎么看出一个票是有散户还是游资还是机构主导的？

## 机构主导特征

### 龙虎榜

- 买入席位主要是机构专用
- 买入金额较大，通常在千万以上
- 买入席位数量较少，但金额集中

### 盘口特征

- 大单持续买入，但不会集中爆发
- 委买盘较大，但不会突然增加
- 分时图走势平滑，波动较小

### K 线特征

- 走势稳健，长期趋势向上
- 不会出现连续涨停
- 在重要位置有明显护盘

### 持仓特征

- 持仓时间较长，不会频繁换手
- 持仓比例较高，通常在 20%以上

## 游资主导特征

### 龙虎榜

- 买入席位主要是知名游资营业部
- 买入金额较大，但可能分散
- 买入席位数量较多

### 盘口特征

- 大单集中买入，成交量急剧放大
- 委买盘突然增加，但可能快速消失
- 分时图波动剧烈，可能出现快速拉升

### K 线特征

- 走势凌厉，可能出现连续涨停
- 波动较大，短期涨幅巨大
- 可能出现快速回调

### 持仓特征

- 持仓时间较短，可能只有几天
- 持仓比例不高，但影响力大

## 散户主导特征

### 龙虎榜

- 买入席位主要是普通营业部
- 买入金额较小，分散
- 买入席位数量很多

### 盘口特征

- 小单交易较多
- 委买委卖盘都不大
- 分时图走势不流畅

### K 线特征

- 走势杂乱，没有明显规律
- 容易受到消息面影响
- 追涨杀跌明显

### 持仓特征

- 持仓时间不确定
- 持仓比例分散
