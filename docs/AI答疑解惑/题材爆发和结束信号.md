# 题材的爆发和结束信号

## 题材爆发信号

### 消息面

- 重大政策利好（如新基建、碳中和等）
- 行业重大事件（如技术突破、产品发布）
- 突发事件（如疫情、自然灾害）

### 市场表现

- 板块内多只股票涨停
- 出现明显的龙头股，连续涨停
- 板块成交量急剧放大
- 板块指数大幅上涨，涨幅居前

### 资金面

- 大资金持续流入
- 龙虎榜显示知名游资或机构买入
- 板块资金净流入排名靠前

### 情绪面

- 市场关注度急剧上升
- 媒体大量报道
- 投资者情绪高涨

## 题材结束信号

### 市场表现

- 龙头股开始走弱，出现跌停或大幅回调
- 板块内股票出现分化，跟风股大幅回调
- 板块成交量萎缩
- 板块指数开始下跌

### 资金面

- 大资金开始流出
- 龙虎榜显示资金大幅卖出
- 板块资金净流出

### 情绪面

- 市场关注度下降
- 投资者情绪转冷
- 出现利空消息或政策变化

### 技术面

- 板块指数跌破重要支撑位
- 板块内多数股票技术形态走坏
