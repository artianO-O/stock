# 竞价凹槽战法

## 概述

竞价凹槽是指个股在集合竞价阶段先高开后回落，形成"凹槽"形态，随后开盘拉升，提供买入机会。

## 核心逻辑

- 集合竞价阶段先高开，显示资金看好
- 随后回落，形成"凹槽"形态，可能是主力洗盘或试探
- 开盘后快速拉升，显示主力拉升意愿
- 市场情绪由分歧转为一致，形成上涨

## 操作要点

### 观察集合竞价

关注集合竞价阶段先高开后回落的个股。

### 识别凹槽形态

竞价阶段形成明显的"凹槽"形态（先高开，后回落，再拉升）。

### 确认拉升信号

开盘后股价快速拉升，突破竞价高点，成交量放大。

### 介入时机

在确认拉升信号后，开盘5-10分钟内及时介入。

### 止损止盈

设置3-5%止损位，目标盈利5-10%。

## 案例

某股集合竞价高开3%，随后回落至平盘附近，形成"凹槽"形态。

开盘后股价快速拉升，突破竞价高点，成交量放大。

按照竞价凹槽战法买入，当日获利5%。

*以上战法仅供参考，实际操作需结合市场环境和个人风险承受能力。投资有风险，入市需谨慎。*
