# 三阳开泰战法

## 概述

股价连续三根小阳线调整，第四天出现放量大阳线，预示后续上涨机会。

## 操作要点

### 识别三阳

出现三根小阳线，成交量相差不大。

### 均线支撑

股价沿五日线运行，不破关键支撑。

### 确认信号

第四天放量拉升，出现大阳线。

### 突破信号

大阳线突破前期高点，信号更可靠。

### 介入时机

在第四天大阳线确认后介入。

## 案例

某股票连续三天收出小阳线，第四天放量大涨，收出大阳线。

随后股价持续上涨，验证三阳开泰战法的有效性。

*以上战法仅供参考，实际操作需结合市场环境和个人风险承受能力。投资有风险，入市需谨慎。*
