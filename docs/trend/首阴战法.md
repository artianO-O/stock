# 首阴战法

## 概述

在上涨趋势中，首次出现阴线调整时，寻找买入机会。

## 操作要点

### 趋势确认

股价处于明显的上升趋势中。

### 识别首阴

首次出现阴线调整，且调整幅度不大（不超过3%）。

### 成交量特征

成交量相对萎缩，显示抛压不重。

### 观察次日

观察次日走势，若止跌企稳，可考虑介入。

### 适用条件

不适合连续阴线的情况。

## 案例

某股票连续上涨5天后，出现一根小阴线，成交量萎缩。

次日股价继续上涨，验证首阴战法的有效性。

---

*以上战法仅供参考，实际操作需结合市场环境和个人风险承受能力。投资有风险，入市需谨慎。*
