# 龙头战法

## 概述

聚焦市场最强题材中的领涨龙头股，追求短期爆发力，遵循"强者恒强"原则。

## 操作要点

### 选择龙头

选择板块内涨停早、封单强、人气高的个股。

### 事件驱动

通常伴随政策或事件驱动。

### 操作周期

操作周期为超短线（1-5天），依赖市场情绪和资金接力。

### 风险控制

设置严格止损，避免追高风险。

### 板块轮动

关注板块轮动，及时切换。

## 案例

某科技股在政策利好下成为板块龙头，连续3日涨停。

短线投资者介入后获利丰厚。

*以上战法仅供参考，实际操作需结合市场环境和个人风险承受能力。投资有风险，入市需谨慎。*
