# 第三阶段：小资金实盘试炼

**学习时间：** 3-6 个月  
**学习目标：** 在真实市场中验证策略，培养实战能力

## 3.1 严控仓位与风险

**仓位管理原则：**

| 账户规模 | 初始投入 | 单股仓位 | 持仓数量 |
| -------- | -------- | -------- | -------- |
| < 10 万  | ≤ 10%    | ≤ 30%    | 1-2 只   |
| 10-50 万 | ≤ 20%    | ≤ 30%    | 2-3 只   |
| > 50 万  | ≤ 30%    | ≤ 25%    | 3-5 只   |

**风险控制规则：**

- **止损：** 单笔亏损不超过本金的 **2%**
- **止盈：** 目标设为 **5%-10%**
- **执行纪律：**
  - 跌破关键支撑位 → 坚决止损
  - 达到目标价 → 坚决止盈
  - ❌ 杜绝死扛亏损
  - ❌ 杜绝贪心恋战

**资金管理：**

- 预留 **30%-50%** 资金应对突发情况
- 不要满仓操作
- 不要借钱炒股

## 3.2 聚焦关键交易时段

**交易时段分析：**

| 时段     | 时间        | 特点               | 操作策略                       |
| -------- | ----------- | ------------------ | ------------------------------ |
| **早盘** | 9:30-10:00  | 波动最大，资金活跃 | 捕捉突破机会，关注高开股       |
| **中盘** | 10:00-11:30 | 相对平稳           | 观察趋势，寻找机会             |
| **午盘** | 13:00-14:30 | 板块轮动明显       | 关注补涨标的，寻找轮动机会     |
| **尾盘** | 14:30-15:00 | 资金博弈激烈       | 规避隔夜风险，不确定时锁定收益 |

**时段操作要点：**

- **早盘：** 重点关注高开股和突破股
- **午盘：** 观察板块轮动，寻找补涨机会
- **尾盘：** 谨慎开新仓，优先处理持仓

## 3.3 坚持每日复盘

**复盘是短线交易进步的关键！必须坚持！**

### 复盘内容清单

**1. 市场整体复盘**

- [ ] 大盘指数走势（上证、深证、创业板）
- [ ] 市场情绪指标（涨跌停数量、连板高度、炸板率）
- [ ] 成交量变化
- [ ] 北向资金流向

**2. 板块复盘**

- [ ] 涨幅榜前 10 的板块
- [ ] 跌幅榜前 10 的板块
- [ ] 板块资金流向
- [ ] 板块内龙头股表现

**3. 个股复盘**

- [ ] 涨停板个股分析（为什么涨停、封板强度）
- [ ] 跌停板个股分析（为什么跌停、是否影响板块）
- [ ] 自选股表现
- [ ] 持仓股分析

**4. 操作复盘**

- [ ] 今日操作记录（买入卖出理由）
- [ ] 操作得失分析
  - ✅ 成功原因：是否遵循策略？是否把握了情绪？
  - ❌ 失败原因：是否违背策略？是否忽视市场情绪？
- [ ] 明日操作计划
- [ ] 需要改进的地方

**复盘模板示例：**

```
日期：2024-XX-XX
市场情绪：□ 高涨 □ 一般 □ 低迷
操作记录：
1. 买入XXX，理由：...
2. 卖出XXX，理由：...
盈亏情况：+XXX / -XXX
明日计划：...
```

**学习检查清单：**

- [ ] 建立并坚持每日复盘习惯
- [ ] 能准确分析操作得失
- [ ] 能通过复盘发现并改进问题
- [ ] 实盘交易至少保持 3 个月不亏损
