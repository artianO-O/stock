# 第二阶段：模拟交易验证策略

**学习时间：** 1-2 个月  
**学习目标：** 在无风险环境中验证交易策略，培养交易节奏

## 2.1 选择平台并熟悉操作

**推荐平台：**

- **国内：** 同花顺模拟炒股、东方财富模拟交易
- **国际：** TradingView（支持多品种）

**必须熟悉的操作：**

- ✅ 下单、撤单
- ✅ 查看盘口（买一卖一、委买委卖）
- ✅ 分析数据（K 线、成交量、技术指标）
- ✅ 设置止盈止损
- ✅ 查看持仓和盈亏

**⚠️ 重要提醒：** 模拟交易要像实盘一样认真对待，避免因操作失误在实盘时亏损！

## 2.2 制定并测试短线策略

**策略构建流程：**

1. **策略设计**

   - 明确买入条件
   - 明确卖出条件
   - 明确止损止盈规则
   - 明确仓位管理规则

2. **策略回测**

   - 使用历史数据验证
   - 统计胜率、盈亏比
   - 分析最大回撤
   - 优化参数设置

3. **策略优化**
   - 调整均线周期
   - 优化 RSI 阈值
   - 改进买入卖出时机
   - 剔除无效策略

**经典策略示例：**

### 早盘竞价战法

- **时间：** 9:15 - 9:25
- **观察要点：**
  - 涨停股封单量
  - 板块热度
  - 高开幅度（3%-5%）
  - 成交量放大
- **买入条件：** 高开 3%-5% + 放量 + 板块强势

### 涨停回调低吸策略

- **关注标的：** 强势股首次回调
- **介入位置：** 5 日、10 日均线附近
- **买入条件：** 缩量回调 + 均线支撑 + 板块未走弱

**学习检查清单：**

- [ ] 能熟练操作模拟交易平台
- [ ] 至少构建并测试 3 种不同策略
- [ ] 理解策略回测的基本方法
- [ ] 能优化策略参数提高胜率

## 2.3 培养交易节奏

**核心要点：**

- **控制交易频率：** 避免高频无效交易
- **适应市场情绪：**
  - 情绪低迷时：减少操作，观望为主
  - 热点明确时：精准切入，快速获利
  - 情绪高涨时：保持警惕，及时止盈
- **形成个人节奏：** 找到适合自己的交易频率和风格

**学习检查清单：**

- [ ] 能在不同市场情绪下调整策略
- [ ] 形成稳定的交易节奏
- [ ] 模拟交易至少保持 1 个月盈利
