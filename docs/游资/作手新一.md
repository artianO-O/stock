# 作手新一

**概述**：知名游资，擅长短线操作。

## 核心心法

### 1. 市场跟随

**核心原则**：跟随市场热点，不做逆势操作

- 跟随市场热点，不做逆势操作
- 顺势而为，不逆势操作
- 市场热点往往具有最强的资金关注度和上涨动力

### 2. 龙头战法

**核心原则**：专注龙头股，利用其强势特征

- 专注龙头股，利用其强势特征
- 龙头股具有最强的市场号召力
- 龙头股往往能带来最大的收益空间

### 3. 风险控制

**核心原则**：设置严格止损，控制回撤

- 设置严格止损，控制回撤
- 严格执行止损纪律，避免大幅亏损
- 控制单笔交易风险，保护本金

### 4. 情绪把握

**核心原则**：理解市场情绪变化，选择合适时机

- 理解市场情绪变化，选择合适时机
- 市场情绪高涨时积极操作，情绪低迷时谨慎观望
- 把握市场情绪转折点，及时调整操作策略

*以上心法仅供参考，实际操作需结合市场环境和个人风险承受能力。投资有风险，入市需谨慎。*
