# 方新侠

**概述**：知名游资，擅长题材炒作。

---

## 核心心法

### 1. 题材挖掘

**核心原则**：提前挖掘市场热点题材

- 提前挖掘市场热点题材
- 关注政策、事件、行业变化等可能引发市场热点的因素
- 提前布局，享受题材炒作带来的溢价

### 2. 龙头操作

**核心原则**：专注板块龙头，享受溢价

- 专注板块龙头，享受溢价
- 龙头股具有最强的市场号召力和资金关注度
- 龙头股往往能带来最大的收益空间

### 3. 情绪周期

**核心原则**：理解情绪周期，选择介入时机

- 理解情绪周期，选择介入时机
- 市场情绪分为冰点、回暖、高潮、退潮四个阶段
- 在不同阶段采取相应的操作策略

### 4. 资金管理

**核心原则**：合理分配仓位，控制风险

- 合理分配仓位，控制风险
- 根据市场情况灵活调整仓位
- 避免重仓单一股票，分散投资

---

*以上心法仅供参考，实际操作需结合市场环境和个人风险承受能力。投资有风险，入市需谨慎。*
