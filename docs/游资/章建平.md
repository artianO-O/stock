# 章建平

**概述**：知名游资，以稳健操作著称。

## 核心心法

### 1. 稳健操作

**核心原则**：注重风险控制，稳健盈利

- 注重风险控制，稳健盈利
- 不追求暴利，追求稳定收益
- 控制回撤，保护本金

### 2. 价值投资

**核心原则**：结合价值投资理念，选择优质标的

- 结合价值投资理念，选择优质标的
- 关注公司基本面，选择有成长性的公司
- 长期持有优质标的，享受公司成长带来的收益

### 3. 资金管理

**核心原则**：严格控制仓位，避免过度杠杆

- 严格控制仓位，避免过度杠杆
- 根据市场情况灵活调整仓位
- 避免重仓单一股票，分散投资

### 4. 长期持有

**核心原则**：对于优质标的，可以长期持有

- 对于优质标的，可以长期持有
- 不频繁交易，减少交易成本
- 享受公司成长带来的长期收益

*以上心法仅供参考，实际操作需结合市场环境和个人风险承受能力。投资有风险，入市需谨慎。*
